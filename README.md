# InShape

InShape - is a full-stack web application for coaches and athletes. After loggin user can search different exercises, create your own workout and take notes. 

## Built with
- ### Frontend
    - React
    - Rest API
    - Styled components
    - React icons
 
- ### Backend
    - Node.js
    - Express.js
    - Uuidv4
    - MongoDb
    - Auth0
